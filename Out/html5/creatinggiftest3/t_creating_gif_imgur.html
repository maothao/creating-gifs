Creating an animated GIF from video format on imgur.com

Imgur.com allows you to upload video URLs directly from YouTube without having to download or convert. You are also able to add text to the GIF.

You will need to have a video file chosen beforehand

  1. Find a video URL from YouTube
  2. Upload it to imgur.com/vidgif
  3. Select the section you want turned into a GIF
  4. You are able to add text in the transparent box that appears on the video
  5. Create GIF
