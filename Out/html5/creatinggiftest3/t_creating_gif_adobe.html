Creating an animated GIF from video file on Abode Photoshop CC

Creating an animated GIF from your own personal video files using Adobe Photoshop is easy and simple. Using Adobe Photoshop gives you more control over editing and to easily add text.

You will need the following before creating an animated GIF through Adobe Photoshop:

  • Video file
  • Adobe Photoshop CC
  1. Have video file available
  2. Open up Adobe Photoshop
  3. Go to File > Import > Video Frames to Layers
  4. Dialogue box will appear. Choose which section of the video file you want to turn into a GIF.
    Can turn entire video into a GIF or can select a section
  5. Go to Timeline view to view each layer the video has been separated into
  6. Once completed, save file to convert into GIF format
