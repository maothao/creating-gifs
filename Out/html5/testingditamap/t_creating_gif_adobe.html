Creating an animated GIF from video file on Abode Photoshop CC

Creating an animated GIF from your own personal video files using Adobe Photoshop is easy and simple. Using Adobe Photoshop gives you more control over editing and to easily add text.

You will need the following before creating an animated GIF through Adobe Photoshop:

  • Video file
  • Adobe Photoshop CC
  1. Open up Adobe Photoshop.
  2. Go to Video Frames to Layers.
    To get to Video Frames to layers, go to File, then Import. This will allow you to select your chosen video.
    A window will appear.
  3. Choose which section of the video file you want to turn into a GIF.
    You can turn entire video into a GIF or can select a section.
  4. Go to Timeline view to view each layer the video has been separated into.
    The Timeline view allows you to see all of the layers your video has been separated into and makes it easier for you to edit either all the layers or only specific ones.
  5. Optional: You can choose to add text to the layers.
    1. Select the layer(s) you want text to appear in in Timeline view and in Layer view.
    2. Select Text Box from tool bar to click and drag over where you want the text to be located on the video layer.
    3. Format text to personal preferences.
      You can change font type, font color, font size, and font orientation.
      You have added text to your GIF.
  6. Once completed, save file by going to Save for web.
    To get to Save for web, go to File. Save file under GIF format.
You have created your GIF.
