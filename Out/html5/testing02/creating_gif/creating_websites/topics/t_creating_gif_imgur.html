Creating an animated GIF from video file on imgur.com

Imgur.com allows you to upload video URLs directly from YouTube without having to download or convert. You are also able to add text to the GIF and once completed, upload your GIF directly to Imgur's search engine type site.

You will need to have a video file chosen beforehand.

  1. Upload video URL to imgur.com/vidgif.
  2. Select the section you want turned into a GIF.
    Refer to the image in Step 5.
  3. You will be able to add text in the transparent box that appears on the video.

  4. Process your GIF by selecting Create GIF.
